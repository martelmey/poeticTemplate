# poeticTemplate
/// Introduction

Lay down bars based on these inputs:

* No. of bars
* No. of measures per bar
* No. of feet per measure
* Type of foot

/// Example input

poetemplate lay-bars {$BAR_COUNT} {$MEASURES_PER_BAR} {$FEET_PER_MEASURE} {$FOOT_TYPE}

poetemplate lay-bars 4 4 5 iamb

/// Example output

Bar count = 4
Measures per bar = 4
Feet per measure = 5
Foot type = iamb
